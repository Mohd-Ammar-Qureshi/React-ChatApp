# 💬🤖 React Chat App with Users & AI Bot

A modern chat application built using React and TailwindCSS that supports multi-user chatting and an AI-powered chatbot.

🌐 Live Demo

https://project-iotrb.vercel.app/

## 🚀 Features

- 👥 Multiple chat users
- 💬 Send and receive messages (in process)
- 🤖 AI Bot for automated replies
- 🗂️ Sidebar with user list
- 🔄 Switch between users
- 💾 Messages stored (LocalStorage / state)
- 🎨 Styled with TailwindCSS
- 📱 Fully responsive UI

## 🛠️ Tech Stack

- React
- Vite
- TailwindCSS
- JavaScript

## 📌 About Project

This project demonstrates how to build a multi-user chat application using React. It includes user switching, message handling, and an AI bot for simulated conversations.

## 🤖 AI Bot

- Auto replies to user messages
- Simulates chatbot interaction
- Can be extended using real AI APIs

## 👥 Chat Users

- Add/remove users
- Switch between chats
- Separate message history per user
